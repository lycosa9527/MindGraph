# ECharts Library

ECharts library is required for the public dashboard map visualization.

## Download

**URL:** https://cdn.jsdelivr.net/npm/echarts@5.5.0/dist/echarts.min.js

**Save as:** `static/js/echarts.min.js`

**Size:** ~800KB minified

**License:** Apache 2.0

## Alternative Download Methods

1. **Using the script:**
   ```bash
   python scripts/download_dashboard_dependencies.py
   ```

2. **Manual download:**
   - Visit: https://echarts.apache.org/zh/download.html
   - Download ECharts 5.x minified version
   - Save to `static/js/echarts.min.js`

**Note:** The dashboard map will not work without this file.
