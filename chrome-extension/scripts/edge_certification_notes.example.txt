OVERVIEW
MindGraph connects Edge to our AI diagram platform. Page content is read only when the user explicitly triggers an action (generate map, extract document, MindMate page context, or Document Summary save). No background monitoring.

REQUIREMENTS
- Edge 120+
- MindGraph account + API token (mgat_…)
- HTTPS to our server

SETUP
1. Sign in at https://test.mindspringedu.com (or https://mg.mindspringedu.com).
2. Web app → Account → API Token → create mgat_ token.
3. Extension → Settings: Server, Account (phone), API token → Save (verifies via GET /api/auth/me).
